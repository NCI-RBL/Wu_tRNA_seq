    Background

    Background¶

    tRNA quantification is hard. cDNA synthesis is hampered by: * pervasive RT blocks by modified (mostly methylated) nucleosides ... this leads to premature RT stops at modified sites... resulting in very short reads. * extensive similarity between tRNA genes ... eg. tRNA genes coding for 2 different codons can have only a single nucleotide difference. ... this make the alignment step of analysis pipeline extremely difficult.

    The composition of cellular tRNA pools is critical for efficient mRNA decoding and proteome integrity. Defective tRNA biogenesis is linked to neurological disorders and cancer.

    AlkB: AlkB is a protein found in E. coli which removes alkylation groups from ssDNA.

    Isodecoder: tRNA genes that share the same anticodon but have different body sequences have been termed as “isodecoders”, implying that they all read the same codon in translation.

    tRNA human genes: Facts:

    • Number of different tRNA anticodon families: 57
    • Number of unique tRNA molecules produced (all cell types included): ~400

    K562: Human CML (chronic myelogenous leukemia) cell line
